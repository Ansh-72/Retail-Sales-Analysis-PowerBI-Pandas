# Retail Sales Analysis (Power BI + Pandas)

## About the Project
This project focuses on analyzing retail sales data using **Pandas** for data cleaning and **Power BI** for data visualization.  
The main goal is to understand revenue, profit, and overall sales performance through an interactive dashboard.

---

## KPIs Included
- Total Revenue  
- Total Profit  
- Profit Margin (%)  
- Total Orders  
- Average Delivery Days  

---

## Tools Used
- Python (Pandas)
- Power BI Desktop
- CSV Dataset

---

---

## How to Use
1. Open the Power BI dashboard file (`.pbix`) using **Power BI Desktop**
2. The dataset used for this analysis is provided in CSV format
3. Data cleaning and preprocessing are done using Pandas

---

## Conclusion
This project shows how raw retail data can be cleaned, analyzed, and visualized to gain meaningful business insights using Power BI.
